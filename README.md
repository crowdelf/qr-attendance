# СИСТЕМА УЧЁТА ПОСЕЩАЕМОСТИ СТУДЕНТОВ С ИСПОЛЬЗОВАНИЕМ QR-КОДОВ



Описание проекта

Данный проект представляет собой веб-приложение для автоматизированного учёта посещаемости студентов с использованием QR-кодов. Система предназначена для упрощения процесса фиксации присутствия студентов при входе в учебное заведение и исключения ручного ввода данных.

Приложение работает полностью в браузере и не требует серверной части. Все данные сохраняются локально с помощью LocalStorage.



Почему этот проект важен

Традиционный учёт посещаемости требует времени и подвержен человеческому фактору. Использование QR-кодов позволяет автоматизировать процесс, снизить вероятность ошибок и ускорить сбор данных. Проект демонстрирует практическое применение современных веб-технологий для решения реальной задачи в образовательной среде.



#### Основные возможности:



1\)Генерация персонального QR-кода на основе ФИО и номера группы студента

2\)Сканирование QR-кода с помощью камеры устройства

3\)Автоматическая фиксация посещения с сохранением даты и времени

4\)Хранение данных о посещениях в LocalStorage браузера

5\)Просмотр журнала всех зарегистрированных посещений

6\)Простой и интуитивно понятный пользовательский интерфейс



#### Используемые технологии:



HTML5;CSS3;JavaScript;QRCode.js (генерация QR-кодов); jsQR (сканирование QR-кодов); LocalStorage



#### Установка и запуск:



1\)Скачайте или клонируйте репозиторий проекта.

2\)Откройте файл index.html в любом современном веб-браузере (Google Chrome, Firefox, Edge).

4\)Для работы сканирования необходимо разрешить доступ к камере устройства.



#### Использование:

1\)Перейдите в раздел «Генерация QR-кода».

Введите ФИО и номер группы студента.

2\)Сгенерируйте QR-код.

3\)Перейдите в раздел «Сканирование» и отсканируйте QR-код.

4\)Просмотрите зафиксированные посещения в разделе «Журнал посещений».



#### Ограничения:

1\)Данные хранятся только локально в браузере

2\)Отсутствует серверная часть и централизованная база данных

3\)При очистке данных браузера информация о посещениях будет удалена
